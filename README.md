# Salama Insurance Portal

A mobile-first insurance application built with Next.js, React, TypeScript, and Tailwind CSS.

## Features

- Mobile-first responsive design
- Web support
- Quote management
- Document upload
- Payment processing
- Policy management

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
